# 🚀 2025 Vibe Coding 期末專案：個人履歷自我介紹網站（Ping-Han Chen）

## 🌟 1. 專題願景 (Vibe)
- **目標對象：** 實習/求職的招募方、想快速了解我的人（主管、面試官、合作夥伴）
- **設計風格：** 日系簡約、乾淨明亮、帶一點水藍＋粉色的柔和氛圍（清楚、好讀、有層次）
- **核心功能：**
  - 用單頁網站快速呈現「自我介紹 / 技能 / 學經歷 / 專案作品 / 聯絡方式」
  - 手機與電腦都能順暢瀏覽（RWD）
  - 點擊 Email 可直接開啟使用者自己的寄信畫面聯絡我（mailto）

## 🤖 2. Gemini 協作紀錄 (Prompt Log)

### ✅ 成功的關鍵提示詞（選 1~2 個）
> **Prompt 1：**  
> 請用繁體中文幫我產出一個單頁式履歷自我介紹網站（Hero/About/Skills/Experience/Projects/Contact），風格日系簡約、乾淨好讀、支援 RWD。請用 Tailwind CDN，區塊使用卡片設計，並加入導覽列可滑到指定區塊。

> **Prompt 2：**  
> 我不想做太複雜的前端互動，請把網站調整成以內容呈現為主，僅保留基本導覽與手機版選單切換即可，並讓視覺層次更清楚。

### 🧩 問題解決紀錄（遇到的困難 & 怎麼請 AI 修）
- **遇到的問題：** AI不認識我 所以我需要一個一個跟他介紹
- **我怎麼問 AI：** (附上我的自我介紹)
- **AI 的建議與修正：**
  - 收到！這份資料非常完整。我已經根據您提供的資訊，將網站內容完全客製化為您的專屬版本。

## 📂 3. 檔案結構說明
- `index.html`：網頁主結構（含 Tailwind CDN 與必要腳本）
- （可選）`style.css`：若未來把 `<style>` 抽離可放這裡
- （可選）`assets/`：放圖片或素材（目前使用線上 avatar 產生器）

## 📈 4. 自我心得與反思
- **使用 Vibe Coding（與 AI 對話寫 Code）的心得：**
  - 我可以把需求用文字講清楚，AI 就能快速產出一個可用的網站雛形，節省大量查資料與排版時間。
  - 在不熟悉前端的情況下，透過「反覆描述我要的效果」也能逐步調整成符合需求的版面。
- **如果可以再優化，我希望增加：**
  - 專案區塊加入 GitHub repo 連結按鈕與更多作品截圖
  - 增加更完整的「下載履歷 PDF」按鈕（或加上英文版切換）
  - 把內容改成更模組化，方便日後維護（例如把 CSS 抽成 style.css）
